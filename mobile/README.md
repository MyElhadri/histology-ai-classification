# Mobile Application

> **Status:** Planned (Phase 3)

This directory will contain the mobile application for medical students.

## Planned Features

- Camera integration for capturing microscope slide images
- Real-time tissue type prediction via API
- Confidence score display
- Educational information panel for each tissue type
- History of previous classifications

## Planned Stack

- **Framework:** Flutter or React Native
- **Communication:** REST API calls to backend service

## Workflow

```
Student opens app → Captures slide image → Sends to API → Receives prediction → Views educational content
```
