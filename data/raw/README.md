# Data — Raw

Original, unmodified dataset files.

## Contents

Place the **Human Histopathological H&E Stained Nuclei Images** dataset here exactly as downloaded from Kaggle.

## Rules

- **Never modify** files in this directory.
- This directory is **gitignored** — data must be downloaded separately by each collaborator.
- Maintain the original directory structure from Kaggle.

## Download

```bash
# Using Kaggle CLI
kaggle datasets download -d <dataset-identifier> -p data/raw/ --unzip
```
