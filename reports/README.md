# Reports

Generated analysis reports, plots, and evaluation metrics.

## Structure

```
reports/
├── figures/    # Saved plots and visualizations (PNG, SVG)
├── metrics/    # Evaluation metrics (CSV, JSON)
└── README.md
```

## Contents

- Training curves (loss, accuracy)
- Confusion matrices
- ROC curves
- Grad-CAM visualizations
- Classification reports
- Per-class performance breakdowns

> Generated files in this directory are **gitignored**. Regenerate by running the evaluation pipeline.
